# resumo-do-lab
Este repositório contém o resumo das lições aprendidas durante o desenvolvimento do lab na DIO
aqui aprendemos a acessar a conta do azure, mudar idioma, mudar recuros da interface do usuario, e navegamos sobre os principais menus do sistema.
Esta lição tratou da Criação de Maquinas Virtuais, MAquinas virtuais em locais distribuidos para melhorar a performance do SLA, mas chamou a atenção ao preço, quanto mais distribuido mais seguraça mas em compesação os valores aumenta por que ao inves de termos uma maquina teremos varias, pou seja se escolhemos 4 regios diferentes de redundancia teremos 4 maquinas a mais os dados sendo replicado em varios locais ao mesmo tempo.
